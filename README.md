Slime Rancher Landing Page

Proyecto de desarrollo web inspirado en el videojuego **Slime Rancher**. 

## Esto es lo que incluye:
* **Estructura:** HTML5 organizado por secciones.
* **Layout:** Uso de **CSS Grid** para la galería de slimes.
* **Diseño:** Degradados dinámicos, fuentes de Google (Fredoka y Quicksand) y efectos de texto.
* **Interactividad:** Botones animados, efectos *hover* en tarjetas y navegación suave (*Smooth Scroll*).

## Hecho con:
* HTML5 / CSS3
* Google Fonts
